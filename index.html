Edge Flipping

Instructions: click on two points to flip the edge between them. It will do nothing if the edge is invalid (i.e. on the hull or if the quad is reflex). When in 'Add Points' mode, click anywhere in the light blue area to add a new point. Note that this will reset the triangulation (and will not keep any changes you've made, even if that's possible in theory). If the current triangulation is Delaunay, a message will display below the graph indicating as such.
